Реле напряжения Devolt 2500 предназначено для индивидуальной защиты приборов от опасных скачков, провалов напряжения и "обрыва нуля" в однофазной сети переменного тока 220В. В случае возникновения аварийной ситуации устройство автоматически отключит питание. После нормализации напряжения, реле автоматически подключит защищаемый прибор.

Особенности:
- цифровая индикация текущего напряжения;
- максимально коммутируемый ток 10А;
- диапазон установки верхнего предела отключения 210-270В (заводская 250В);
- диапазон установки нижнего предела отключения 120-190В (заводская 175В);
- диапазон установки задержки повторного включения 10-600сек;
